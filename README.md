#  Quiz Arena - Multiplayer Quiz Game

Real-time multiplayer quiz battle game.

##  Live Demo

**Play Now:** [https://ashraful2512.github.io/Online-Quiz-Game/](https://ashraful2512.github.io/Online-Quiz-Game/)

##  Features

- Real-time multiplayer battles (up to 8 players)
- 4 game modes: Classic, Speed Run, Survival, Category Master
- Live leaderboard and scoring system
- Room codes for easy sharing
- Speed bonuses for fast answers

##  Quick Start

Open the live demo in multiple browser tabs to test multiplayer!

##  Tech Stack

- **Backend:** Node.js, Express, Socket.io
- **Frontend:** React 18, Socket.io-client
- **Deployment:** GitHub Pages + Render

##  Setup Locally

```bash
# Install dependencies
cd server && npm install
cd ../client && npm install

# Start server
cd server && npm start

# Start client
cd client && npm start
```

##  Game Modes

- **Classic:** Mixed difficulty, standard scoring
- **Speed Run:** Fast-paced with extra speed bonuses
- **Survival:** Elimination mode - wrong answers eliminate players
- **Category Master:** Focus on a single category

##  How to Play

1. Create a room or join with a code
2. Wait for players to join
3. Host starts the game
4. Answer questions quickly for bonus points
5. View live leaderboard after each round
6. Winner is crowned at the end
